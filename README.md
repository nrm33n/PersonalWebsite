# Portfolio website 
Built with Tailwind, React.js, and Material UI
__________________________

# Setup
1. Install dependencies 
```
npm install 
```
or
```
yarn install
```

2. Start development server 
```
npm start
```
or 
```
yarn start
```

# Run production
```
npm run build
```
or 
```
yarn build
```
